Create a Persistent Arch Linux Bootable USB with Vagrant

Thursday, Jan 9, 2020

Preface When installing a linux distribution, it is common for the instructions to have the user create a bootable USB, boot from the device, and proceed with the installation procedure from that live medium. However, this blog post will outline an alternative approach where a virtual machine created with Vagrant will be used in lieu of the live medium. The original intention was to use Docker for this process – leveraging the --device flag and mounting the target USB device in the Docker container, but the underlying hypervisor in Docker Desktop for Mac does not support this.
